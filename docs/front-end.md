## Project Structure

| Project | Contents | Dependencies |
|--|--|--|
| Core | Types, Interfaces, Models | - |
| Components | Directives, Components | Core |
| Api | Entities, Api Services | Core |
| Web | Pages | Core, Components, Api |
| Web-e2e | e2e tests for Web | Web |
